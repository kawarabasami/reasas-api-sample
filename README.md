# ReasasApiSample

RESAS APIの利用サンプル。  
都道府県ごとの総人口推移を折れ線グラフで表示する。  

## ローカルビルド、起動方法

node.jsがインストール済みの端末に本リポジトリをクローン後、下記コマンド押下.  
```
npm install
npm start
```
  
`Compiled successfully`が表示後、下記URLから起動.  
 http://localhost:4200/
